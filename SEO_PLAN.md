## 十、自有电商深度方案

### 为什么鉴定站 + 电商是最高利润模型

```
搜索「ロレックス スーパーコピー 見分け方」的人
→ 已经知道仿表存在
→ 正在研究仿表的质量
→ 处于购买决策链的末端
→ 只要给他一个购买入口，转化率极高
```

这不是冷流量，是**热到发烫的购买意图流量**。月搜 80,000 次的背后是 80,000 个想买仿表的日本人。

### 架构设计（合法分离）

```
┌─────────────────────┐     ┌─────────────────────┐
│ rolex-lab (鉴定站)    │────→│ shop.jp (电商站)      │
│                     │     │                     │
│ 教育/知识           │     │ 产品/交易            │
│ 公开信息            │     │ 独立域名             │
│ 工厂对比            │     │ 代引き決済            │
│ 社区UGC            │     │ 日本仓发货           │
│                     │     │                     │
│ Yahoo收录 ✅        │     │ Cloudflare 全防御    │
│ 无法律风险           │     │ 域名轮换机制          │
└─────────────────────┘     └─────────────────────┘
```

两个站完全独立：
- 鉴定站永不卖货，纯内容，过 AdSense + Yahoo 收录
- 电商站藏在 Cloudflare 后面，被封立刻换域名
- 两个站之间用**软链接**（文章里自然提到，不是硬广告）
